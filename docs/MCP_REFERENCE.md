# MCP Quick Reference

One-page command reference for axiOS MCP integration.

## Quick Commands

### mcp-gateway REST API

```bash
# Check gateway health
curl -s http://localhost:8085/health

# List all MCP servers
curl -s http://localhost:8085/api/servers | jq

# List all available tools
curl -s http://localhost:8085/api/tools | jq

# Count tools
curl -s http://localhost:8085/api/tools | jq 'length'

# Execute a tool
curl -s -X POST http://localhost:8085/api/tools/github/search_repositories \
  -H "Content-Type: application/json" \
  -d '{"query": "axios"}'
```

### mcp-gw CLI (provided by mcp-gateway)

mcp-gateway installs `mcp-gw` for CLI-based on-demand tool discovery. See the mcp-gateway repository for full usage details.

```bash
# List all MCP servers
mcp-gw

# Search for tools by name
mcp-gw grep "file"

# Get specific tool schema
mcp-gw github/create_repository

# Execute a tool
mcp-gw github/search_repositories '{"query": "axios"}'
```

## Available MCP Servers

| Server | Purpose | Setup |
|--------|---------|-------|
| **time** | Date/time utilities | None |
| **github** | GitHub API access | `gh auth login` |
| **journal** | systemd logs | None |
| **context7** | Library documentation | None |
| **axios-ai-mail** | AI-powered email | PIM module |
| **mcp-dav** | Calendar/contacts | PIM module |
| **brave-search** | Web search | API key |

## Service Management

```bash
# Check mcp-gateway status
systemctl --user status mcp-gateway

# View logs
journalctl --user -u mcp-gateway -f

# Restart after config changes
systemctl --user restart mcp-gateway
```

## Configuration Files

Generated by mcp-gateway module:

| File | Purpose |
|------|---------|
| `~/.mcp.json` | Claude Code MCP configuration |
| `~/.config/mcp/mcp_servers.json` | mcp-gateway configuration (used by mcp-gw) |
| `~/.gemini/settings.json` | Gemini CLI configuration |
| `~/.config/ai/prompts/axios.md` | System prompt for AI agents |

## Common Use Cases

### Debugging

```bash
# Check system logs via API
curl -s -X POST http://localhost:8085/api/tools/journal/query_logs \
  -H "Content-Type: application/json" \
  -d '{"unit": "nginx"}'
```

### Development

```bash
# Search GitHub repositories
curl -s -X POST http://localhost:8085/api/tools/github/search_repositories \
  -H "Content-Type: application/json" \
  -d '{"query": "nix MCP"}'
```

### Research

```bash
# Query library docs
curl -s -X POST http://localhost:8085/api/tools/context7/query \
  -H "Content-Type: application/json" \
  -d '{"library": "nixpkgs", "query": "how to override package"}'

# Web search
curl -s -X POST http://localhost:8085/api/tools/brave-search/web_search \
  -H "Content-Type: application/json" \
  -d '{"query": "NixOS modules"}'
```

## Verification

```bash
# Check gateway is running
curl -s http://localhost:8085/health

# List servers
curl -s http://localhost:8085/api/servers | jq '.[].name'

# Verify system prompt is injected
grep -q "axios" ~/.claude.json && echo "Enabled" || echo "Not found"
```
